About The Algorithm

Kruskal's Algorithm is used to find the minimum spanning tree of a connected and undirected graph. A spanning tree of that graph is a sub graph that is a tree and connects all the vertices together. This is done as follows:

  1. Sort all the edges in non-decreasing order of their weight.

  2. Pick the smallest edge. Check if it forms a cycle with the spanning tree formed so far. If cycle is not formed, include this edge. Else, discard it.

  3. Repeat step #2 until there are (V-1) edges in the spanning tree.

The algorithm is a Greedy Algorithm. The Greedy Choice is to pick the smallest weight edge that does not cause a cycle in the MST constructed so far.
